# Ai-power-resume-screening-system
PROBLEM STATEMENT
TRADITIONAL RESUME SCREENING IS TIME–CONSUMING AND PRONE TO HUMAN BIAS.
RECRUITERS STRUGGLE TO FILTER HUNDREDS OF RESUMES QUICKLY AND ACCURATELY.
THERE’S A NEED FOR AN INTELLIGENT, AUTOMATED, AND SCALABLE SYSTEM TO ANALYZE A RESUME BASED ON JOB-SPECIFIC CRITERIA.
Project objectives
Build a web-based system for automated resume classification.
Use NLP and ML models to extract and evaluate key candidate skills
Enable HR to upload a resume and get job-fit predictions instantly
Ensure secure user authentication and result logging
